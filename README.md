# Puget Experience
This is a web app for the University of Puget Sound's Admissions department.
## Installation

Run the following commands to install the app

1. Install dependencies
`bundle install`

2. Set up the database
`rake db:create db:migrate db:seed`

3. Start the server
`rails s`

## Running the tests

`rspec spec/`
